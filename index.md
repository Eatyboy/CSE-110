### Favorite Programming Language
My favorite programming language is C because:
1. The language itself is **simple**
2. It's lack of guardrails makes it **powerful**
3. It's easy to **understand** what C code actually does on a machine

As Linus Torvalds put it,
> "If you think like a computer, writing C actually makes sense."

Some of the ways I've used C include:
- Making a simulation of a double pendulum
- Recreating the popular 2048 game
- Implementing quaternion rotations

Here's a classic hello world in C
```
#include <stdio.h>

int main(void) {
    printf("Hello, world!\n");
    return 0;
}
```
